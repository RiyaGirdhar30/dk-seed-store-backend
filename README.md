# рҹ“– Project Overview

The DK Seed Store Backend provides a secure and scalable REST API for the DK Seed Store mobile application.

It handles user authentication, product management, shopping cart operations, wishlist functionality, order management, payment processing through Razorpay, image uploads using Cloudinary, notifications, addresses, and other backend services.

The backend is built using **Node.js**, **Express.js**, **MongoDB Atlas**, **Mongoose**, **JWT Authentication**, **Cloudinary**, and **Razorpay**, and is deployed on **Render**.

# рҹҡҖ Installation

## Clone the repository

```bash
git clone https://github.com/RiyaGirdhar30/dk-seed-store-backend.git
```

## Navigate to the project

```bash
cd dk-seed-store-backend
```

## Install dependencies

```bash
npm install
```

## Start the development server

```bash
npm run dev          #Development
```

## Start the production server

```bash
npm start            #Production
```
